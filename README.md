## コマンド
### ローカルブランチの作成

```
git checkout -b <ブランチ名>
```

### ローカルブランチをリモートに登録

```
git push -u origin <ブランチ名>
```

### ローカルブランチをリモートと紐づけ
- 該当ブランチにcheckout

```
git branch -u origin/master
```

### ローカルブランチのマージ
- 更新したいブランチにcheckout

```
git marge <更新元のブランチ名>
```

### 現在のブランチを確認

```
git branch --contains
```
